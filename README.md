# CA03-Decision_Tree_Census_Data-
This assignment builds and evaluates a Decision Tree classifier to predict whether a person's income is greater than $50K using census demographic data, including data quality analysis, preprocessing, model training, hyperparameter tuning, and visualization of the best-performing tree.
# Decision Tree Income Classification

## Overview
This project implements a Decision Tree classifier to predict whether a person's income is greater than $50K using demographic information from a census dataset. The workflow includes data quality analysis, data cleaning, model training, hyperparameter tuning, model evaluation, and visualization of the best-performing decision tree.

## Dataset
The dataset comes from the Census Bureau and contains demographic attributes related to individuals and their income category.

Key characteristics:
- Target classes: >50K and <=50K
- Number of attributes: 7
- Number of observations: 48,842

The dataset is loaded directly from the following source:

https://github.com/ArinB/MSBA-CA-03-Decision-Trees/blob/master/census_data.csv?raw=true

Some continuous variables in the dataset are discretized into categorical groups using a process called binning.

## Methodology

### 1. Data Quality Analysis
The dataset is analyzed to identify:
- Missing values
- NaN values
- Potential data inconsistencies

Descriptive statistics and a Data Quality Report are created.

### 2. Data Cleaning and Transformation
Based on the data quality analysis:
- Necessary cleaning steps are performed
- Data transformations and encoding are applied where needed

### 3. Decision Tree Model
A DecisionTreeClassifier from scikit-learn is used to build the classification model. The dataset is split into training and testing sets based on the dataset indicator column.

### 4. Model Evaluation
The model performance is evaluated using:
- Confusion Matrix
- Accuracy
- Precision
- Recall
- F1 Score

### 5. Hyperparameter Tuning
Four hyperparameters are manually varied to find the best-performing model:

- Split Criterion (Entropy or Gini)
- Minimum Samples per Leaf
- Maximum Features
- Maximum Depth

Model performance is recorded and compared to determine the optimal hyperparameter combination.

### 6. Decision Tree Visualization
The best-performing decision tree is visualized using GraphViz to better understand the decision rules generated by the model.

### 7. Prediction
The final trained model is used to predict the income category for a new individual based on provided demographic attributes.

1. Clone or download this repository.
